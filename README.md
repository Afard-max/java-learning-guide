# ☕ Java 21 LTS — Backend Learning Guide

A fully interactive landing page built with **React + Vite** that serves as a visual roadmap for learning Java 21 backend development from absolute zero.

## 🚀 Live Demo

> [[GitHub Pages link here](https://afard-max.github.io/java-learning-guide/)]

## 📋 Overview

This project presents a structured, 5-phase learning path covering everything from Java fundamentals to full CI/CD pipelines with Docker and GitHub Actions. Designed for complete beginners who have never written a line of code.

## 🗺️ Learning Phases

| Phase | Topic                                 | Duration    |
| ----- | ------------------------------------- | ----------- |
| 0     | Absolute Fundamentals + JVM Internals | 0–1 month   |
| 1     | Java Language Basics                  | 0–1 month   |
| 2     | Maven & Project Structure             | 1–3 months  |
| 3     | Spring Boot REST APIs                 | 3–6 months  |
| 4     | Full Project + Docker & CI/CD         | 6–12 months |

## 🛠️ Tech Stack

- **React 18** — UI framework
- **Vite** — Build tool & dev server
- **Pure CSS-in-JS** — No external UI libraries
- **Google Fonts** — Fraunces + JetBrains Mono

## ✨ Features

- Cursor-following ambient glow effect
- Collapsible phase cards with smooth animations
- Scroll-triggered fade-in animations
- Sticky navbar with smooth section navigation
- Scroll-to-top floating button
- Fully responsive layout

## 📦 Getting Started

```bash
# Clone the repository
git clone https://github.com/YOUR_USERNAME/YOUR_REPO_NAME.git

# Navigate to the project
cd YOUR_REPO_NAME

# Install dependencies
npm install

# Start the dev server
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

## 🏗️ Build for Production

```bash
npm run build
```
